# Timezone bot

This bot helps you when you want to convert a time to different timezones.

## Config file

```json
{
  "prefix": "<prefix>",
  "token": "<token>"
}
```

## Setup

```sh
# builds bot to ./build
tsc
# Runs bot with node
node ./build/bot.js
```

## Usage

To display commands type "!tz_help"
